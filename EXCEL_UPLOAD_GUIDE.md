## Excel Format Example

### To UPDATE an existing poster:

| Title | Description | Type | Image URL | Website URL |
|-------|-------------|------|-----------|-------------|
| Ep Release | New description here | poster | https://your-new-image-url.jpg | https://your-website.com |

### To UPDATE an existing video:

| Title | Description | Type | Video URL | Video Thumbnail |
|-------|-------------|------|-----------|-----------------|
| Blacmelo | Updated video | video | https://youtu.be/xxxxx | https://your-thumbnail-url.jpg |

### To CREATE a NEW poster:

| Title | Description | Type | Image URL | Website URL |
|-------|-------------|------|-----------|-------------|
| My New Poster | Brand new poster | poster | https://new-image-url.jpg | https://website.com |

## Important Notes

- **Title must match exactly** (case-insensitive, but spelling must be exact)
- **Type must be**: poster, video, or website
- **For posters**: Include `Image URL` and optionally `Website URL`
- **For videos**: Include `Video URL` and `Video Thumbnail`
- **For websites**: Include `Website URL` and optionally `Description`
- **To create new items**: Use a title that doesn't exist in the list above

## Column Names (Case-Sensitive in Excel)

Use these exact column headers in your Excel file:

- Title
- Description
- Type
- Image URL
- Video URL
- Video Thumbnail
- Website URL

- title
